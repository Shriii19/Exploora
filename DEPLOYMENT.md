# Deployment Checklist

## Pre-Deployment

- [ ] Update all URLs in sitemap.xml with actual domain
- [ ] Update robots.txt with actual domain  
- [ ] Create social media images (1200x630px for OG, 1200x675px for Twitter)
- [ ] Add favicon files (favicon.svg, favicon.ico)
- [ ] Test all pages on mobile devices
- [ ] Verify all forms work correctly
- [ ] Check all internal links

## SEO Configuration

- [ ] Submit sitemap to Google Search Console
- [ ] Submit sitemap to Bing Webmaster Tools
- [ ] Verify site ownership in search consoles
- [ ] Add Google Analytics tracking code
- [ ] Test social media card previews
- [ ] Verify structured data with Google Rich Results Test

## Performance

- [ ] Optimize and compress all images
- [ ] Minify CSS and JavaScript files
- [ ] Enable gzip compression on server
- [ ] Configure browser caching headers
- [ ] Test page load speed with Google PageSpeed Insights

## Security

- [ ] Add SSL certificate (HTTPS)
- [ ] Configure security headers
- [ ] Set up CORS if needed
- [ ] Review and update Content Security Policy

## Post-Deployment

- [ ] Test all functionality on production
- [ ] Monitor error logs
- [ ] Check analytics are tracking
- [ ] Verify search engines are crawling
- [ ] Test contact form submissions
