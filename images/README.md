# Images Folder Structure

This folder should contain all social media images for SEO optimization.

## Required Images

### Open Graph Images (1200x630px)
- `og-image.jpg` - Homepage
- `destinations-og.jpg` - Destinations page
- `blog-og.jpg` - Blog page
- `about-og.jpg` - About page
- `contact-og.jpg` - Contact page
- `planner-og.jpg` - Trip Planner page
- `destination-og.jpg` - Destination Detail page

### Twitter Card Images (1200x675px)
- `twitter-image.jpg` - Homepage
- `destinations-twitter.jpg` - Destinations page
- `blog-twitter.jpg` - Blog page
- `about-twitter.jpg` - About page
- `contact-twitter.jpg` - Contact page
- `planner-twitter.jpg` - Trip Planner page
- `destination-twitter.jpg` - Destination Detail page

### Logo
- `logo.png` - Site logo (transparent background, 512x512px recommended)

## Image Guidelines

### Open Graph (Facebook/LinkedIn)
- Dimensions: 1200x630 pixels
- Format: JPG or PNG
- Max file size: 8MB
- Aspect ratio: 1.91:1

### Twitter Cards
- Dimensions: 1200x675 pixels
- Format: JPG or PNG
- Max file size: 5MB
- Aspect ratio: 16:9

### Best Practices
- Use high-quality, relevant images
- Include branding elements
- Ensure text is readable
- Test on dark and light backgrounds
- Optimize file sizes for fast loading
